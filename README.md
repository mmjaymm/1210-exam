## Requirements
- PHP >= 7.2.5
- BCMath PHP Extension
- Ctype PHP Extension
- Fileinfo PHP extension
- JSON PHP Extension
- Mbstring PHP Extension
- OpenSSL PHP Extension
- PDO PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension.

## Step to Install

```
1. git clone https://github.com/mmjaymm/1210-exam.git
2. cp .env.example .env
3. change database settings under .env
4. composer install
5. php artisan migrate
6. npm install
7. npm run dev  
```
